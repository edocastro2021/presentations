# Monitoreo en tiempo real de datos geoespaciales con Microsoft Fabric

## 🎯 Objetivo
Explicar cómo implementar un sistema de monitoreo geoespacial en tiempo real usando Microsoft Fabric.

## 🧱 Arquitectura
- Event Streams
- KQL Database
- Lakehouse
- Power BI

## 📂 Contenido
- slides/: presentación usada en la charla
- notebooks/: demos en Fabric
- scripts/: creación de tablas, pipelines, etc.
- architecture/: diagramas

## 🚀 Cómo ejecutar el demo
1. Crear Lakehouse en Fabric
2. Ejecutar scripts en /scripts
3. Cargar datos de ejemplo
4. Ejecutar notebook
